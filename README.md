Padee is a Flutter mobile application designed to function as a digital bead counter for prayer and meditation.
It is built using the GetX micro-framework for state management and routing.

## Screenshots

## Features

- **Bead Types:** Choose from three bead count types: 9, 27, or 108.
- **Round Counting:** Automatic round counting with individual bead count reset.
- **Goal Setting:** Set goals based on the number of rounds you wish to complete.
- **Goal Alerts:** Receive notifications upon goal completion via pop-up, sound, and vibration.
- **Localization:** Supports both English and Myanmar languages.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (latest version recommended)
- Dart SDK

### Running the App

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app on your connected device or emulator.
